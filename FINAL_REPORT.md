## 最終リサーチレポート

## 1. エグゼクティブサマリー

2019年開店・足立区西保木間のミュージックバーropeの歴史ドキュメント化を目的とした6ステップリサーチを実施した。確定事実は5点（開店年・所在地・マスター名・営業継続・主要常連）にとどまり、ドキュメント本体の約80%は取材未実施による未収集状態にある。一方で、14ページ・クロニクル年表型の構成設計、執筆ガイドライン、取材チェックリストの整備が完了しており、「器」は出来上がっている。唯一かつ最大のボトルネックは**たいぞう君へのインタビュー実施**であり、それさえ完了すれば完成度の高いドキュメントが見込める。

## 2. 調査の背景と目的

### 2.1 プロジェクト概要

| 項目 | 内容 |
|------|------|
| 対象 | 西保木間ミュージックバーrope（2019年開店、継続営業中） |
| 目的 | マスターたいぞう君と常連客の歩みを、地域文脈を交えて歴史ドキュメントとして記録・物語化する |
| 主読者 | 常連客・音楽ファン・地域コミュニティ |
| トーン要件 | 親しみやすく、ユーモアを交えた歴史ドキュメント |

### 2.2 リサーチ設計

以下の6ステップ構成で調査を設計・実施した。

```
Step1: ミュージックバー文化・日本の小規模ライブバー歴史の背景整理
Step2: 2019年前後の地域音楽シーン・西保木間エリアの特色調査
Step3: 開店エピソード・年表の詳細化（一次情報最優先）
Step4: 常連キャラクター（岡ちゃん・マイコちゃん）エピソードの物語化
Step5: 年表・転換点・アーティスト記録の統合とスライド構成化
Step6: 批評的レビュー・読者視点での品質確認
```

## 3. 主要な発見事項（重要度順）

### 発見①：ドキュメントの80%は取材なしに執筆不可能【最重要】

rope固有の情報はオンライン上に公開資料がほぼ皆無であり、LLMの学習データにも含まれていない。確定事実5点のみでは年表・人物紹介・エピソード章はすべて空白のままとなる。

> **primary_source率：25%（4件中1件）／tier1ソース：0件**

この数値は「もっと調べるべき」という曖昧な認識を、「取材なしにはドキュメントの成立が不可能」という具体的な行動指針へと変換した。

### 発見②：演出先行バイアスの検出と排除【構造的発見】

初期調査では「どう見せるか（演出・メタファー体系）」の設計が「何を見せるか（事実収集）」に先行するという典型的バイアスが発生していた。Critic Reviewによりこれを早期検出し、**取材先行型への方針転換**を確定した。

具体的に排除が確定した要素：

| 要素 | 問題 | 対応 |
|------|------|------|
| Campbell英雄神話論の適用 | 小規模コミュニティの物語に不釣り合い | **不採用確定** |
| 「rope＝繋がりの縄」メタファー体系 | 店名由来未確認のまま構築 | **凍結確定** |
| 具体的シーン描写・冒頭コピーの創作 | 一次情報なしの演出先行 | **禁止確定** |

### 発見③：確証バイアスの具体的所在【品質管理】

以下3点の解釈先走りが検出され、取材時のオープンクエスチョン化という対策を設定した。

- 岡ちゃんの充電切れ＝「居心地の良さの証明」という解釈（本人確認なし）
- 「伝説のブルースマン」という称号の根拠未確認（実績ベースか愛称ベースか不明）
- 新規読者への過剰配慮による、主読者（常連）の満足度低下リスク

### 発見④：「器」の完成【実務的成果】

取材結果を流し込む受け皿として、以下の成果物を整備した。

- ✅ 14ページ・クロニクル年表型の目次構成（柔軟調整可能な設計）
- ✅ 人物紹介テンプレート・年表テンプレート（取材チェックリスト兼執筆フレーム）
- ✅ 執筆ガイドライン（ハルシネーション防止ルール・トーン設定・情報源区別）
- ✅ 取材必要項目リスト（8カテゴリ・17問）

### 4.3 矛盾の解消

#### 矛盾①：「ハルシネーション回避」と「魅力的な物語化」

MUST制約（架空エピソード創作禁止）とWANT制約（人間味ある物語性）は原則として競合する。**「編集」と「創作」の明確な線引き**により解消した。

| 許可される「編集」 | 禁止される「創作」 |
|-----------------|-----------------|
| 取材事実の時系列整理・配置設計 | 取材で未確認のセリフや心情の推測的記述 |
| たいぞう君・常連の言葉の直接引用 | 存在確認されていないイベントの記述 |
| 背景情報の補足（「当時の足立区では〜」等） | 断片情報の積極的解釈による場面創作 |

グレーゾーンは「〜という趣旨のことがあったという」と伝聞・再構成を明示する。

#### 矛盾②：「rope」メタファー体系の先走り

店名由来が未確認のため、既存のメタファー体系は凍結。取材で由来が判明した後に、**その由来に即したメタファー体系を新たに設計**する（既存体系の流用は不可）。由来が「特に深い意味はない」場合は、メタファーに依存しない構成に切り替える。

#### 矛盾③：岡ちゃんの解釈先走り

「伝説のブルースマン」が実績ベースか愛称ベースかにより、人物紹介のトーンが根本的に異なる。取材確認後に以下を適用する。

- 実績ベース → 音楽歴・代表的演奏を具体的に記述
- 愛称ベース → ropeコミュニティの「キャラ立ち」文化の一例として紹介
- 両方の場合 → 実績と親しみを両立した記述

## 5. 結論と提言

### 5.1 最大ボトルネック：たいぞう君インタビューの実施

ドキュメント完成への唯一のボトルネックは取材実施である。以下に優先度順のインタビューガイドを示す。

#### 第1ブロック：核心情報（これがないとドキュメントが成立しない）

1. 店名「rope」の由来
2. ropeを開いたきっかけ・動機。なぜ西保木間だったのか
3. たいぞう君の音楽歴（楽器・バンド・好きなアーティスト・音楽との出会い）
4. 開店日のエピソード（2019年何月開店か・最初のお客さんは誰か）
5. コロナ禍（2020〜2022年頃）の乗り越え方・一番きつかった時期

#### 第2ブロック：年表・記録

6. 7年間で特に印象に残っているライブ・イベント・出来事（年ごとに）
7. 出演アーティストで特に印象に残っている人
8. ropeの音楽ジャンルの傾向
9. キャパシティ・ステージ形態・PA設備などの規模感

#### 第3ブロック：人物・コミュニティ

10. 岡ちゃんの人物像・ブルースマンとしての演奏・充電切れの詳細
11. マイコちゃんとropeとの出会い・印象的なエピソード
12. 他に紹介したい常連やエピソード
13. ropeにとって「常連」とはどういう存在か

#### 第4ブロック：哲学・未来

14. ropeを続けてきて、変わったこと・変わらないこと
15. ropeが地域や音楽シーンに果たす役割についての自己認識
16. これからのropeへの夢・目標・やりたいこと
17. 来てくれるお客さんへ伝えたいこと

> **取材上の留意点：** 質問はオープンエンドで（「〜について聞かせてください」形式）。たいぞう君の言葉をそのまま記録（録音推奨）。想定と異なる回答こそ歓迎し、演出フレームに合わない事実がドキュメントを本物にする。

### 5.2 取材後の完成ロードマップ

```
Phase 1: 取材実施
         たいぞう君 → 岡ちゃん・マイコちゃん → 現地撮影
         ↓
Phase 2: 取材情報の整理・年表作成・事実確認
         ↓
Phase 3: 目次構成の最終調整（取材結果に基づきページ配分確定）
         ↓
Phase 4: 各章の執筆（LLMが構成・文章化を支援、一次情報を最優先）
         ↓
Phase 5: 常連によるレビュー（内輪エピソードの正確性チェック）
         ↓
Phase 6: 最終仕上げ・プレゼンテーション化
```

### 5.3 LLMの役割の明確化

本プロジェクトにおけるLLMの正しい役割は以下に限定される。

| LLMが担う役割 | LLMが担わない役割 |
|-------------|----------------|
| 構成・テンプレート設計 | rope固有の事実の生成 |
| 取材情報の文章化・編集支援 | 年表・エピソードの創作 |
| 背景情報の補足（地域・時代文脈） | 本人未確認の心情・解釈の提示 |
